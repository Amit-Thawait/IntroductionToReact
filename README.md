# IntroductionToReact
=======
This is a basic repository for getting started with React js application.

You need to have node(version 5.1.1) installed on you machine.

To setup:
-----------
```
npm install -g webpack
```

```
npm install 
```

To run :
-----------

```
webpack -w
```

```
npm start
```

To run tests:
--------------
```
npm test
```
